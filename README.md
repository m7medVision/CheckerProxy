# Checker Proxy
## Install packages:
use this command to install packages
```
pip install -r requirements.txt
```
## run the bot:
* put you proxies in ```proxies.txt``` then run the script.
use this commands to run script:
```
python CheckerProxy.py
```
or
```
python3 CheckerProxy.py

```
or
```
py3 CheckerProxy.py
```
or 
```
py CheckerProxy.py
```
